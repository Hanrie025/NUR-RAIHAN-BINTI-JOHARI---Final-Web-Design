# Personal Resume Website

This is my personal resume website built using HTML and CSS.  
It showcases my skills, experience, education, and contact information.

## Pages
- **index.html** – Home page
- **biodata.html** – About Me / Biodata
- **career.html** – Career Overview
- **contact.html** – Contact Information

## Features
- Responsive layout
- Simple and clean design
- Navigation menu linking all pages
- Images included in the `images` folder

## Instructions
- Open `index.html` in a web browser to view the website.
- All CSS is included inside the HTML `<style>` tags.
